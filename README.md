# Myshop - CRA Typescript / Redux

## Tech stack

-   React
-   Redux toolkit
-   React router
-   Typescript
-   React Helmet

## Installation

Install dependencies

`yarn` or with npm `npm install`

## Available Scripts

### `yarn start`

### `yarn test`

### `yarn build`

### `yarn eject`

## Important Notice

Hostname for development server is set to `shop.local`, make sure to bind it to your localhost
